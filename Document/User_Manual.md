# User Manual
## Rental Management System

### 1. Getting Started
#### 1.1 Login
- Access the mobile app or web portal.
- Enter your username and password.
- Click **Login**.

### 2. For Property Owners
#### 2.1 Adding a Property
1. Navigate to "My Properties".
2. Click "+" button.
3. Enter Name and Location.
4. Save.

#### 2.2 Creating a Contract
1. Go to "Contracts" section.
2. Select "New Contract".
3. Choose Tenant and Apartment.
4. Set Start/End Dates (System will validte availability).
5. Confirm.

### 3. For Tenants
#### 3.1 Paying Rent
1. Navigate to "Payments".
2. Select your active contract.
3. Choose Payment Method (Online/Cash).
4. Submit.

#### 3.2 Requesting Maintenance
1. Go to "Maintenance".
2. Describe the issue.
3. Submit Request.
